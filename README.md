# contactor_picker

联系人选择器

## Getting Started

```
ContactorPicker.showPicker(
                    context,
                    ///数据源
                    dataList: [],
                    ///选中数据
                    onSelectedData: (data) {

                    },
                    ///标题
                    title: '联系人',
                    ///页面背景色
                    backgroundColor: Color(0xFFFAFAFA),
                    ///当前字母颜色
                    letterSelectedColor: Colors.blueAccent,
                  );
```
